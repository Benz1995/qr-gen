# Agent Skills ที่แนะนำสำหรับโปรเจกต์นี้

เอกสารนี้สรุป skills ที่เหมาะกับการพัฒนาโปรเจกต์ **เว็บ Tool ทราฟฟิกสูง (Free + SEO + Ads)**

## Skills ที่ติดตั้งให้แล้ว

1. `doc`
   - ใช้จัดเอกสารสเปก, requirements, launch checklist ให้ทีมทำงานต่อได้ไว
2. `playwright`
   - ใช้ทดสอบ flow หน้าเว็บแบบ end-to-end อัตโนมัติ
3. `screenshot`
   - ใช้จับภาพหน้าจอ feature ก่อน/หลังเปลี่ยนสำหรับรีวิวงาน
4. `security-best-practices`
   - ใช้ตรวจ baseline ด้านความปลอดภัยของเว็บและ API
5. `vercel-deploy`
   - ใช้ช่วยเตรียม/ตรวจขั้นตอน deploy เว็บ Next.js ขึ้น Vercel

## ใช้ skill ไหนเมื่อไหร่

- ตอนสรุปงาน/เขียนเอกสาร: `doc`
- ตอนตรวจ flow สำคัญ (generator, auth, dashboard): `playwright`
- ตอนส่งงาน UI หรือแก้หน้าเว็บ: `screenshot`
- ตอนใกล้ขึ้น production: `security-best-practices`
- ตอนจะปล่อยจริง: `vercel-deploy`

## หมายเหตุ

หลังติดตั้ง skill ใหม่ ปกติควร restart Codex เพื่อให้ระบบหยิบใช้ skill ได้ครบถ้วน
